Social Media Feed App

A minimalist social media feed built with vanilla JavaScript, HTML, and CSS. The app displays user posts featuring avatars, locations, images, comments, and likes. Users can toggle the like state by clicking the heart icon, which updates the like count dynamically.

Features

- Dynamic rendering of posts from a data module
- Like button with toggle and live count update
- Semantic HTML with accessibility in mind (`alt` tags)
- Responsive layout with clean, modern design
- Font Awesome icons for UI elements
- Modular ES6 JavaScript using event delegation

Disclaimer

This project is intended for educational purposes only. It is a simplified simulation of a social media feed and does not connect to any real backend or store user data. Use it as a learning tool to understand front-end concepts, DOM manipulation, and event handling.

No personal data is collected or transmitted. Use responsibly.
