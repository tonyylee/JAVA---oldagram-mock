# Static Social Media Feed (HTML & CSS)

A static recreation of a social media feed built using only HTML and CSS. This version focuses on layout, spacing, typography, and visual hierarchy without any JavaScript or interactivity.

## Features

- Semantic HTML structure
- CSS Flexbox layout
- Responsive design principles
- Custom typography using Google Fonts
- Clean and organized styling

## Purpose

This project was created to practice front-end layout and styling fundamentals before adding JavaScript-based interactivity.

## Technologies Used

- HTML5
- CSS3
- Google Fonts

## Disclaimer

This project is for educational purposes only. It is a static UI mockup and does not include any functionality, data storage, or backend integration. No user data is collected or processed.
